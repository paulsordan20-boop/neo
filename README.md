# Neo Valentine — Personal Valentine Website

Open `neovalentine.html` in a browser to view the page.

Features:
- Vintage polaroid-style layout
- Upload photos and videos to the gallery
- Compose and save a message shown on the hero card
- Save Valentine plans (stored in `localStorage`)
- Nursing quiz: 5 questions, 30s per question

No build tools required — just open the HTML file.
